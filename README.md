# VisionCraft 🎨

A premium, bento-grid text-to-image application powered by the Hugging Face FLUX.1 model.

## 🚀 Deployment

This project is optimized for **Vercel**. To deploy:

1. Import this repository to [Vercel](https://vercel.com/new).
2. Add the following **Environment Variable**:
   - `VITE_HF_TOKEN`: Your Hugging Face API key.
3. Deploy!

## ✨ Features

- **FLUX.1 Model:** High-quality image generation via the new Hugging Face Unified Router.
- **Bento Grid Layout:** Premium, responsive design with glassmorphism effects.
- **Prompt Ideas:** Quick-start chips for Cyberpunk, Ethereal, and Sci-Fi prompts.
- **Responsive:** Optimized for all screen sizes.

## 🛠️ Local Setup

1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with `VITE_HF_TOKEN="your_key_here"`.
4. Run `npm run dev`.
